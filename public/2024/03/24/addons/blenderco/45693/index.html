1.6 版

2023-11-10

  • 添加 ToggleRegion 工具
    • 单个键盘映射，可根据鼠标在屏幕上的位置切换工具栏、侧边栏和资产架/浏览器TT-PanelN-Panel
      • 注意：新的资源架在 Blender 4.0 中还不是很有用，因此默认情况下禁用切换，但如果需要，可以启用和使用
      • 更有趣的是在 3D 视图的顶部或底部切换一个新的资源浏览器区域
        • 它将记住之前选择的库和catalog_id以及所有资源浏览器设置
          • 例如thumbnail_size、import_method、过滤器选择和过滤器搜索词，以及标题对齐方式
        • 之前设置的资源浏览器区域高度也会被记住
        • 而所有这些工作空间！
          • 因此，在每个工作区上，您可以拥有 2 个不同的资产浏览器，它们可以整齐地隐藏起来或在需要时显示
        • 然后，不仅可以在 3D 视图中使用鼠标关闭资源浏览器，还可以将鼠标放在资源浏览器的中心，就好像它是一个区域，而不是一个区域
          • 将鼠标移向资源浏览器的两侧，您也可以切换资源库/目录选择或活动资源详细信息，也只需按一下T
    • 注意：此工具现在在MACHIN3tools中默认启用
    • 另外：检查插件首选项，了解您可以进行的一些行为调整，并确保观看有关该工具的简短视频，因为一些 Blender 恶作剧
  • 底纹饼
    • 在场景级别存储每个着色类型的叠加可见性
    • 向插件首选项公开默认的覆盖可见性设置
    • 处理不同步的叠加可见性设置，如果用户使用其他方式切换叠加，则可能会发生这种情况
    • 斜角着色器
      • 添加选项（默认为“打开”）以考虑网格尺寸以调制斜角着色器
        • 这样可以防止小物体出现较大的斜角
        • 在此基础上，您仍然可以选择调节每个对象
      • 补偿对象缩放
    • 阴影平滑/平整工具
      • 修复了 4.0 由于斜角粗细和折痕 API 更改而导致的问题
  • 保存馅饼
    • 截屏视频
      • 固定垂直定位
    • 支持 .stl 导入/导出，默认禁用
  • 视图饼
    • SmartView摄像头
      • 添加轮询以确保它仅在对象模式下运行
  • 收藏馅饼
    • 删除自定义 AddToCollection（） 和 MoveToCollection（） 操作
      • Blender 的原生操作现在可以在本地视图中正常工作，即使没有大纲上下文覆盖
  • 光标和原点饼图
    • 使用“原点到活动/选择”按钮将原点切换到光标的位置
      • 光标现在在右边，其他光标按钮在上面的饼图中
  • Filebrowser/Asset浏览器工具
    • 支持在系统文件浏览器中使用 Key 从 assetbrower 中打开资源文件夹，就像在 filebrowser 中一样O
    • 尝试顶部打开资产的 .blend 文件时显示淡入淡出警告（如果该文件已打开）现在已打开
  • CreateAssemblyAsset 工具
    • 修复并简化 Blender 4.0 中的缩略图分配
  • 智能边缘
    • 修复了 4.0 由于斜角粗细和折痕 API 更改而导致的问题
    • 修复 KnifeProject Cut Through 错别字
  • 镜像工具
    • 简化在 DM 贴花上移动正常传输模组
  • 材料拾取器工具
    • 检查资产浏览器是否存在时静音警告
  • 自定义工具
    • 将“导航首选项”重命名为“输入和导航”
      • 执行NDOF设置
    • 将NFOF_BUTTON_FIT view_selected键盘映射更改为NDOF_BUTTON_1 view_center_pick
    • 禁用 shift + B zoom_border 键盘映射
    • 将curve.select_linked键盘映射从 CTRL + L 更改为 SHIFT dbl LEFTMOUSE
    • 在 Blender 4.0 中，禁用（需要）alt_navigation，用于在对象、网格和曲线模式下平移、旋转和调整大小操作
  • SmartVert 工具
    • 幻灯片扩展
      • 支持 HyperCursor 0.9.15 以及以前的预发布版本
  • 插件首选项
    • 将插件类别更改为 3D 视图
    • 引入registration_debug设置来控制是否将（取消）注册确认写入终端
    • 简化和重构各种设置的绘制方式
    • 修复了插件键盘映射部分中禁用所有键映射工具或饼图时未显示警告提示的问题
    • 将 CURVEmachine 和 HyperCursor 链接添加到“关于”部分
  • 组上下文菜单
    • 强制INVOKE_DEFAULT执行上下文以支持从键盘映射调用此 （sub） 菜单
  • 全面支持Blender 4.0
    • 将 dpi arg 放在 blf.size（） 中
    • 支持新的着色器名称
    • 处理与资产浏览器相关的 API 更改
  • 修复较新 numpy 版本的 np.float 问题
  • 静音 reload_modules（） 调试输出
